# FTPServer - 轻量级 Windows 平台 FTP 服务器

## 🚀 产品简介

FTPServer 是一款专为 Windows 平台打造的高性能、轻量级 FTP 服务器软件。采用原生 C++ 开发,支持完整的 FTP 协议标准,提供稳定可靠的文件传输服务。无论是个人文件共享、团队协作,还是企业内部数据交换,FTPServer 都能为您提供安全、高效的解决方案。

## ✨ 核心特性

### 1. 🌐 完善的 FTP 协议支持
- **双模式传输**: 支持主动(PORT)和被动(PASV)两种传输模式,完美适配各种网络环境
- **全功能命令集**: 支持 20+ 标准 FTP 命令,涵盖下载、上传、删除、重命名等所有常用操作
- **IPv4/IPv6 双栈**: 自动枚举网卡智能识别 IPv4 和 IPv6 地址,支持多网卡环境

### 2. 🔒 安全可靠
- **身份认证**: 支持用户名/密码认证登录,也可配置匿名访问
- **权限控制**: 5 大操作权限(下载、上传、删除、重命名、创建目录)独立配置,灵活可控
- **路径安全**: 内置路径遍历保护,禁止访问根目录之外的文件
- **传输完整性**: 自动校验传输文件大小,确保数据完整无缺

### 3. 🎯 完美支持中文
- **双编码模式**: 支持 UTF-8 和 GBK 双编码,彻底解决中文文件名乱码问题
- **宽字符 API**: 全面采用 Windows 宽字符 API,原生支持中文路径和文件名
- **智能兼容**: 自动适配旧版和新型 FTP 客户端编码需求

### 4. ⚡ 高性能传输
- **大缓冲区优化**: 64KB 数据缓冲区 + 256KB TCP 缓冲区,大幅提升大文件传输速度
- **多线程处理**: 独立线程处理每个客户端连接,支持高并发访问
- **磁盘空间检查**: 自动检查可用磁盘空间,避免传输中断
- **断线防护**: 完善的 socket 管理和错误处理,防止资源泄漏

### 5. 🖥️ 友好的用户界面
- **图形化配置**: 直观的 GUI 界面,鼠标操作即可完成所有配置
- **系统托盘**: 最小化到系统托盘,不占用任务栏空间
- **开机自启**: 支持开机自动启动,持续提供 FTP 服务
- **网卡智能识别**: 自动显示所有可用网卡,一键切换监听地址

### 6. 📋 完善的日志系统
- **四大日志分类**: 错误日志、操作日志、访问日志、传输日志
- **详细记录**: 记录每次登录、文件传输、目录操作等信息
- **一键打开**: 系统托盘菜单一键打开日志文件夹,方便查看

## 📚 支持的 FTP 命令

| 命令 | 功能描述 |
|------|----------|
| USER | 用户名登录 |
| PASS | 密码验证 |
| QUIT | 断开连接 |
| SYST | 查询系统类型 |
| TYPE | 设置传输类型 |
| PWD | 显示当前目录 |
| CWD | 切换目录 |
| CDUP | 返回上级目录 |
| OPTS UTF8 | 设置 UTF-8 编码 |
| PORT | 主动模式传输 |
| PASV | 被动模式传输 |
| LIST/DIR/LS | 列出文件 |
| SIZE | 获取文件大小 |
| RETR | 下载文件 |
| STOR | 上传文件 |
| DELE | 删除文件 |
| RMD | 删除目录 |
| MKD | 创建目录 |
| RNFR/RNTO | 重命名文件 |
| MDTM | 获取修改时间 |
| NOOP | 空操作(保持连接) |

## 🔧 配置选项

- 📁 **访问目录**: 自定义 FTP 根目录
- 🔌 **监听端口**: 自定义 FTP 服务端口 (1-65535)
- 🌐 **网络适配器**: 选择监听网卡 IP 地址
- 👤 **用户认证**: 配置用户名和密码
- 🔐 **访问权限**: 独立控制 5 大操作权限
- 💬 **欢迎消息**: 自定义登录欢迎语
- 👋 **退出消息**: 自定义退出提示语
